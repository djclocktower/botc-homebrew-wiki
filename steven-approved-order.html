Steven Approved Order

The canonical ordering for homebrew character abilities

Steven Approved Order

    Within any group (e.g. "If you die"), abilities are sorted by ability text length from least to greatest number of characters.

    If there is a tie, they are then sorted by character name from least to greatest number of characters.

    If, miraculously, there is a tie again, they are sorted alphabetically.
