How do I get a token?
  1. GitHub → your profile → Settings
  2. Developer settings → Personal access tokens → Fine-grained tokens
  3. Generate new token. Repository access → Only select repositories → botc-homebrew-wiki
  4. Permissions → Contents → Read and write
  5. Generate, copy it, paste above. It saves in this browser so you only do this once.
