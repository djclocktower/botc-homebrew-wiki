How do I get a token?
  1. GitHub → Settings → Developer settings
  2. Personal access tokens → Fine-grained tokens → Generate new token
  3. Repository access → Only select repositories → botc-homebrew-wiki
  4. Permissions → Contents → Read and write, then generate & paste above.
