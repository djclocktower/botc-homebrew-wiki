Standardize character icons

Re-fits every published character's art onto the standard 591×591 frame: it trims the empty transparent space around each figure, then scales and centers it so every icon appears at a consistent size — matching the official wiki. The original art is replaced with the standardized version in storage; the site updates as soon as the run finishes.

Before you start: make sure the wiki is unlocked (a lock blocks uploads). This changes art for every character at once. It's safe to re-run, and you can cancel at any time. Nightly backups keep 30 days of history if you ever need to compare.
