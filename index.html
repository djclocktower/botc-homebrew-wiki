About This Wiki

The BOTC HomeBrew Wiki is a community-maintained library of fan-made characters for Blood on the Clocktower. Every character page includes full almanac content — summary rules, How to Run instructions, examples, jinxes, and an official-schema JSON export ready to drop into the script tool.

Anyone can contribute a character using the built-in authoring tool. Browse the existing characters for inspiration, and credit goes to the original creator on every page.

Fan-made content. Not affiliated with The Pandemonium Institute. For the official game, rules, and merchandise, visit bloodontheclocktower.com.
